# 指令字典 V1.0

## 一、大厅指令（未连接状态）
由大厅处理模块本地处理，不产生网络流量。

| 指令 | 参数 | 说明 |
| ---- | ---- | ---- |
| /create <房间名> <房主ID> <端口> | 房间名：UTF-8，1~32字符，含空格需引号包裹；房主ID：[A-Za-z0-9_]，1~16字符；端口：1~65535 整数 | 在本机启动服务器进程，生成管理令牌（本地注入客户端缓存及服务器启动参数），并自动以房主身份连接。 |
| /join <你的ID> <服务器地址> | 你的ID：[A-Za-z0-9_]，1~16字符；服务器地址：主机:端口，如 192.168.1.5:8080 | 连接到目标聊天室。连接成功后进入已连接状态。 |
| /config <设置项> <值> | 设置项：port、nickname、roomname、max_users；值：根据设置项校验合法性 | 修改本地默认配置并持久化。 |
| /help [指令名] | 指令名：可选 | 无参数显示所有大厅指令简介；带参数显示该指令详细语法。 |
| /quit | 无 | 退出整个程序。 |

## 二、聊天室通用指令（已连接状态）
以 / 开头，由指令路由模块处理。本地执行或发往服务器。

| 指令 | 参数 | 执行位置 | 说明 |
| ---- | ---- | ---- | ---- |
| /help [指令名] | 指令名：可选 | 本地 | 无参数显示所有聊天室指令简介；带参数显示该指令详细语法。 |
| /clear | 无 | 本地 | 清空本地消息窗口屏幕。 |
| /myid | 无 | 本地 | 显示自己在聊天室中的实际ID（从本地缓存读取）。 |
| /list | 无 | 服务器 | 列出在线用户。普通用户看到ID列表；房主看到 [短ID] ID 映射。 |
| /msg <用户ID> <消息> | 用户ID：目标用户；消息：文本 | 服务器 | 发送私聊消息。目标不在线则返回错误。 |
| /quit | 无 | 本地 + 服务器 | 退出聊天室，本地断开连接，服务器检测后广播离线。客户端返回未连接状态。 |

## 三、房主管理指令（已连接状态，仅房主可用）
发往服务器处理。客户端自动附加管理令牌，服务器校验令牌和权限。客户端本地做第一道权限拦截。

| 指令 | 参数 | 说明 |
| ---- | ---- | ---- |
| /kick <用户ID> | 用户ID：完整ID 或 短ID（数字） | 强制断开指定用户，广播被踢消息。 |
| /mute <用户ID> | 用户ID：完整ID 或 短ID | 禁言指定用户。被禁言者发送聊天消息会被服务器拒绝。 |
| /unmute <用户ID> | 用户ID：完整ID 或 短ID | 解除禁言。 |
| /lock | 无 | 锁定聊天室，拒绝新成员加入。广播锁定通知。 |
| /unlock | 无 | 解锁聊天室，恢复新成员加入。广播解锁通知。 |

## 四、指令语法通用规则
- 所有指令以 / 开头，后接小写指令名。
- 指令名与参数之间、参数与参数之间以一个或多个空格分隔。
- 最后一个参数若包含空格，客户端解析时将剩余部分整体作为该参数（如 `/msg Bob 你好 世界` → 目标 Bob，消息 你好 世界）。
- 含空格参数也可使用双引号包裹（如 `/create "我的房间" Alice 8080`）。
- 参数校验失败（缺失、格式错误、越界），立即返回错误提示，不发送网络数据。
- 服务器指令执行失败（权限不足、目标不存在等），返回 `TYPE_RESPONSE` 并附带错误描述。

## 五、权限与安全

| 机制 | 说明 |
| ---- | ---- |
| 管理令牌 | 房主创建聊天室时由客户端本地生成，通过启动参数注入服务器。房主客户端发送管理指令时自动附加令牌，服务器校验。令牌永不通过网络传输。 |
| 客户端第一道防线 | 指令路由模块检查本地缓存的 is_host 标记，非房主输入管理指令时本地直接拒绝。 |
| 服务器第二道防线 | 指令处理器校验令牌和房主身份，客户端伪造标记无法绕过。 |
| 锁定机制 | 房主可通过 /lock 禁止新连接加入，用于配合 /kick 防止被踢用户立即重连。 |

## 六、错误提示规范

| 场景 | 提示内容（示例） |
| ---- | ---- |
| 未知指令 | 未知指令：/xxx。输入 /help 查看可用指令。 |
| 参数缺失 | /kick 需要指定用户ID。输入 /help kick 查看用法。 |
| 参数格式错误 | 端口号必须为 1~65535 之间的整数。 |
| 权限不足（本地拦截） | 权限不足：/kick 仅房主可用。 |
| 权限不足（服务器返回） | [系统] 指令执行失败：权限不足。 |
| 目标不存在 | [系统] 用户 'Bob' 不在线。 |
| 自己被禁言 | [系统] 您已被禁言，无法发送消息。 |
| 发送频率过高 | [系统] 操作过于频繁。 |
