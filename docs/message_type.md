```
用户输入
  │
  ▼
是否以 "/" 开头？
  │
  ├── 否 ──> 打包为 CHAT 类型，发送给服务器
  │
  └── 是 ──> 提取命令词
              │
              ▼
         查找本地指令字典
              │
              ├── 未命中 ──> 提示“未知指令”，不发送任何网络数据
              │
              └── 命中 ──> 是否是本地指令？
                            │
                            ├── 是 ──> 本地执行（如 /quit、/clear、/help）
                            │
                            └── 否 ──> 打包为 COMMAND 类型，发送给服务器
```

我们已经完成了协议处理器的设计。接下来建议：

把公共实体（Message、MessageType、自定义异常）快速过掉，因为它们最简单。

设计设置模块和在线用户表，它们也是底层依赖。
